ToDo app 


 
Product idea: 

An application where we can store To do items as a list. Being able to add new items, remove items and edit them. The data should be stored in a database.	 

 

User Stories:

As a user, I want to add tasks to the list so that I can check what I need to do. 
As a user, I want to delete the task, so that I can remove the tasks that are not needed anymore. 
As a user, I want to mark the task as done, so that I know the tasks that are completed.  
As a user, I want to edit the tasks, so that I can make changes to the task if needed. 
As a user, I want the tasks to be automatically loaded when the application starts, so that I can know the tasks that are already there in the application. 
As a user, I want to create a new task with title and description, so that I know what the task is about. 

 

Product backlog:

Design the database 
Scaffold the database 
Design the appearance of the application windows 
Code for reading the data from database and showing items in a listbox 
Functionality for adding a new ToDo item 
Functionality for removing a ToDo item 
Functionality for editing items and marking them as done
